# 夜间自动迭代任务配置

## 权限说明

夜间任务执行时，自动授予以下权限：

1. **文件修改权限** - 允许自动修改 `backend/`、`frontend/`、`scripts/` 目录下的代码文件
2. **测试执行权限** - 允许运行 `npm run test` 和 `python -m pytest`
3. **Git 操作权限** - 允许执行 `git add`、`git commit`、`git push`

## 执行规则

- ✅ 代码审查后自动应用简单修复
- ✅ 测试失败时自动尝试修复
- ✅ 重复代码自动提取为公共函数
- ✅ 测试通过后自动提交变更
- ⚠️ 复杂重构仍需人工批准

## 通知方式

任务执行完成后，通过飞书发送执行报告到指定频道。
